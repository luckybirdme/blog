vuejs 插件

插件通常用来为 Vue 添加全局功能。
常用插件包括 vue-router, vuex 等

一，插件功能：

1. 添加 Vue 函数的全局方法或者属性。如: vue-custom-element

2. 添加 Vue 实例方法，在 Vue.prototype 上实现。

3. 添加全局资源：指令/过滤器/过渡等。如 vue-touch

4. 通过全局混入来添加一些组件选项。如 vue-router

二，插件开发
