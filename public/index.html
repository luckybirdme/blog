python list tuple dict set

list 是一种有序的集合，可以随时添加和删除其中的元素，存放不同数据类型。
tuple 是另一种有序的列表，与 list 非常类似，但是创建完毕后就不能修改了。
dict 是一种 key-value 保存的无序集合，查询速度快，消耗内存也多。
set 是一种去重式的 key 无序集合，类似没有 value 的 dict 。
